2. Use Cases – Simplified Version
Actor: User (Guest).
Use Case: Completing a Quick Quiz
• Prerequisites: None (the application is accessible to everyone upon visiting the site).
• Step 1 (Selection): On the main page, the user selects the technology (Spring/Angular) and defines the question pool (10/20/30).
• Step 2 (Question Session): The system draws the appropriate number of questions. The user answers subsequent questions while viewing their progress on the progress bar.
• Step 3 (Completion): After the final question, the system calculates the result.
• Final State: The user sees a summary screen with their score.
