Use Cases (Przypadki Użycia)
Use cases opisują główne scenariusze z aktorami, pre/post warunkach i krokami. Podzielone na role.

User (Użytkownik)
Rozpoczęcie Quizu: Pre: Zalogowany. Kroki: Wybiera quiz → Widzi listę pytań → Odpowiada zaznaczając opcję → Submit. Post: Wyniki zapisane.

Sprawdzenie Wyników: Pre: Po quizie. Kroki: Przegląda dashboard → Widzi procent poprawnych, błędy z wyjaśnieniami.

Śledzenie Postępu: Kroki: Loguje się → Otwiera dashboard → Widzi wykresy postępów, leaderboards.

Admin (Twórca/Admin)
Dodanie Quizu: Pre: Zalogowany jako admin. Kroki: Idzie do panelu → Tworzy quiz (temat, liczba pytań) → Dodaje pytania/opcje/poprawne odpowiedzi.

Edycja Pytania: Kroki: Wybiera quiz → Edytuje/edytuje pytanie → Zapisuje.

Analiza Wyników Użytkowników: Kroki: Otwiera raporty → Widzi statystyki quizów, słabe obszary.
